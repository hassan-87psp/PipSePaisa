--------------------------------------------------------
STEP 2 — ONESIGNAL REST API KEY lo
--------------------------------------------------------
OneSignal dashboard -> Settings -> Keys & IDs ->
"REST API Key" copy karo. (Ye SECRET hai, kabhi HTML me mat
daalna.)

--------------------------------------------------------
STEP 3 — SUPABASE me EDGE FUNCTION banao
--------------------------------------------------------
Supabase Dashboard -> Edge Functions -> "Deploy a new function"
- Name: notify-signal
- Code: is zip ki file
      supabase/functions/notify-signal/index.ts
  ka SAARA code copy-paste karo -> Deploy.

Function settings me: "Verify JWT" ko OFF karo
(Settings -> Enforce JWT verification = OFF).

--------------------------------------------------------
STEP 4 — SECRETS add karo (function ke liye)
--------------------------------------------------------
Supabase -> Edge Functions -> Manage secrets (ya Settings ->
Edge Functions -> Secrets) me 2 secrets:

  ONESIGNAL_REST_API_KEY  =  (Step 2 wali key)
  HOOK_SECRET             =  koi bhi random text, e.g. pip_9f3k2x

--------------------------------------------------------
STEP 5 — DATABASE WEBHOOK banao (signal aate hi function chale)
--------------------------------------------------------
Supabase -> Database -> Webhooks -> "Create a new hook"
- Name: signal-push
- Table: signals
- Events: Insert  (sirf Insert tick karo)
- Type: HTTP Request
- Method: POST
- URL: https://vjqvoinsspgsrcyhwspy.supabase.co/functions/v1/notify-signal
- HTTP Headers (Add 2):
      x-hook-secret : pip_9f3k2x     (jo HOOK_SECRET Step 4 me diya)
      Content-Type  : application/json
- Save.
